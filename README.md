## Systematic Sampling

This R Shiny application demonstrates:
- Sample size determination for population mean
- Incorporation of subgroup structure
- Bias constraint
- Cost and time optimization
- Systematic sampling within subgroups
- Visualization of experimental design

### Methodology
- Sample size derived using variance-based estimator with bias control
- Neyman allocation adjusted for cost and time
- Systematic sampling applied independently in each subgroup
